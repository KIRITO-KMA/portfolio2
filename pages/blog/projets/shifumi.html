Shifoumi : Pierre-Feuille-Ciseaux

Un jeu classique de Pierre-Feuille-Ciseaux avec trois niveaux de difficulté et des animations fluides pour une expérience de jeu engageante.

Présentation du Projet

Shifoumi est une implémentation moderne du jeu classique Pierre-Feuille-Ciseaux, développé en HTML, CSS et JavaScript pur. Le projet met l'accent sur une expérience utilisateur fluide avec des animations attrayantes et trois niveaux de difficulté pour s'adapter à tous les joueurs.

Contexte

Ce projet a été développé pour démontrer mes compétences en développement web front-end, en particulier dans la manipulation du DOM, la gestion des événements et les animations CSS. Le jeu a été conçu pour être à la fois amusant et techniquement intéressant, avec une attention particulière portée à l'expérience utilisateur.

Fonctionnalités

  • Trois niveaux de difficulté (Facile, Moyen, Difficile) avec des IA différentes
  • Animations fluides pour les interactions utilisateur
  • Effets sonores pour une expérience immersive
  • Design responsive qui s'adapte à tous les appareils
  • Compteur de points avec fin de partie à 3 victoires
  • Effets visuels pour les victoires, défaites et égalités

Technologies utilisées

HTML5 CSS3 JavaScript (ES6+) Tailwind CSS Font Awesome Web Audio API

Captures d'écran

Détails techniques

Le jeu utilise une approche orientée objet en JavaScript pour gérer la logique du jeu. Les principaux défis techniques relevés incluent :

  • Implémentation d'une logique avec différents niveaux de difficulté
  • Gestion des animations et des transitions CSS pour un rendu fluide
  • Utilisation de l'API Web Audio pour les effets sonores
  • Optimisation des performances pour une expérience de jeu réactive

Le code est entièrement commenté et structuré de manière modulaire pour une meilleure maintenabilité.
