Test de Réflexe : Jeu de réflexe

Un jeu de réflexe avec souris pour une expérience de jeu engageante.

Présentation du Projet

Test de Réflexe est un jeu de réflexe avec souris, développé en HTML, CSS et JavaScript pur. Le projet met l'accent sur une expérience utilisateur fluide avec une animation attrayante et un meuilleur score pour appater les joueurs.

Contexte

Ce projet a été développé a mes heure perdu pour mon portfolio et garder la main sur le JavaScript.

Fonctionnalités

  • Un bouton pour commencer le jeu
  • Un effet sonore pour chaque situation
  • Un changement de couleur en cas de victoire ou defaite
  • Une possibilité de rejouer et de battre son record
  • Une possibilité de mettre en plein ecran

Technologies utilisées

HTML5 CSS3 JavaScript Web Audio API Font Awesome

Captures d'écran

Détails techniques

Le jeu utilise une approche orientée objet en JavaScript pour gérer la logique du jeu. Les principaux défis techniques relevés incluent :

  • Implémentation d'une logique avec différents niveaux de difficulté
  • Gestion des animations et des transitions CSS pour un rendu fluide
  • Utilisation de l'API Web Audio pour les effets sonores
  • Optimisation des performances pour une expérience de jeu réactive

Le code est entièrement commenté et structuré de manière modulaire.
