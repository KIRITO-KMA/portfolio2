Présentation de la formation

Le BTS Services Informatiques aux Organisations option SLAM est une formation de niveau Bac+2 qui forme des spécialistes du développement d'applications. La formation allie enseignements théoriques et pratiques avec une forte dimension professionnelle.

Compétences acquises

Développement

  • Développement web (HTML5, CSS3, JavaScript, PHP, Python)
  • Programmation orientée objet (Java, PHP)
  • Bases de données (SQL, MySQL, MariaDB)
  • Frameworks (CodIgniter, ...)

Gestion de projet

  • Méthodes agiles (Scrum, Kanban)
  • Gestion de versions (Git, GitLab, GitHub)
  • Rédaction de cahiers des charges
  • Tests et documentation technique

Projets réalisés

Application de gestion de parc informatique

Technologies : PHP, MySQL,codeIgniter

Description : Développement d'une application web pour la gestion du parc informatique du lycée, permettant le suivi des équipements, des interventions et des demandes de maintenance.

Site e-commerce PermaBook

Technologies : WordPress, MySQL

Description : Conception et développement d'une plateforme e-commerce complète avec gestion des utilisateurs, catalogue produits, panier et système de sauvegarde.

Application mobile de suivi sportif

Technologies : Java, MySQL

Description : Développement d'une application mobile permettant de suivre ses performances sportives, avec synchronisation des données et tableaux de bord personnalisés.
